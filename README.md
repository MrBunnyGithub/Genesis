# Genesis

What is Genesis?

Genesis is a multisig dapp. Similar to Gnosis.

Step 1. 

Download Repository or git clone https://github.com/MrBunnyGithub/Genesis.git

Step 2.

```

cd path/to/Genesis/folder && npm install && cd app && npm install
cd ..
npx hardhat compile
cd app
npm start

```
